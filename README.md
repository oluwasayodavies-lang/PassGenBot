# 🔐 PassGen Bot - Telegram Password Generator

A secure, easy-to-use Telegram bot for generating strong passwords instantly.

## ✨ Features

- Generate passwords of custom length (4-64 characters)
- Preset lengths: 8, 12, 16, 20, 24, 32 characters
- Strength indicator for each password
- Secure generation - passwords never stored
- User-friendly interface with inline buttons
- Responsive design for all devices

## 🚀 Quick Deploy

### Deploy on Railway

1. Fork/clone this repository
2. Go to https://railway.app
3. Click "New Project" → "Deploy from GitHub repo"
4. Select your repository
5. Add environment variable:
   - Key: `BOT_TOKEN`
   - Value: `your_telegram_bot_token_here`
6. Deploy!
